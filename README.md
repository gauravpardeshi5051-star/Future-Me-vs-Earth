# Future-Me-vs-Earth

Hi, this is my project ‘Future Me vs Earth’.
This app shows how our daily lifestyle impacts the future of Earth.

As I select options, and move to 2050, you can see Earth’s condition change.

I generate AI insights using Google Gemini…

It gives a personalized future scenario and suggestions.

The goal is to make sustainability emotional and actionable.

Note: API keys are securely handled using environment variables.
